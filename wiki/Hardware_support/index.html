This section contains information on common hardware issues and offers troubleshooting tips and corrective measures.

Components:
  • Head
  • Arms
  • End Effectors
Diagnostic Steps:
  • If you suspect there is a problem with a hardware component on Baxter, use the Field Service Menu (FSM) on the robot to run hardware diagnostic tests on the hardware and JCB controller boards.
  • You can also use the Robot Monitor tool to view hardware-related errors over ROS.

Need more help? For Hardware-related issues with your Baxter, please contact the Rethink Support team at: support@rethinkrobotics.com
